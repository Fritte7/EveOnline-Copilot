EVE Online Copilot

It's a small tracker with a Terminal look & feel to track the character in wormhole space.
Fun little project with small easter egg to have fun and learn navigation compose.

Future feature:
- Historic - Timeline
- Detail for wormhole
- Favourite and notification that you reach the system
- TTS
- KMP introduced to manage the app in desktop mode

OAuth with SSO

Koin as DI
MVVM architecture
Navigation Compose
Coroutines with Flow
Room
